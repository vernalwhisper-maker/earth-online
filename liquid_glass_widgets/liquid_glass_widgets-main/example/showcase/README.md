# Showcase App

Example demonstrating `liquid_glass_widgets` in a luxury travel app.

## Features

- Quality inheritance patterns with `AdaptiveLiquidGlassLayer`
- Centralized theme system for consistent glass styling
- Real-world UI examples: navigation, forms, booking flows

## Running

```bash
cd showcase
flutter pub get
flutter run
```

## Assets Required

Place 6 destination images (1080x1440px minimum) in `assets/images/`:
- `santorini_villa.jpg`
- `zermatt_chalet.jpg`
- `marrakech_riad.jpg`
- `borabora_bungalow.jpg`
- `nyc_penthouse.jpg`
- `bali_treehouse.jpg`
