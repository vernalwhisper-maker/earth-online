# 地球Online v1.2.0 调试功能需求日志

> 创建日期: 2026-07-21
> 涉及模块: MoreSettingsPage.jsx / DebugPage.jsx / App.jsx / settingsStore.js

---

## 需求列表

### 1. 隐藏式开发者模式触发
- **触发位置**: 设置 -> 更多设置 -> 二级页面"更多设置"标题
- **触发方式**: 600ms 内连续点击标题 7 次
- **视觉效果**: 触发后在页面底部出现"高级调试"卡片

### 2. 高级调试卡片控制 (标题行开关)
- 卡片标题"高级调试"右侧有一个开关
- 触发后此开关默认为开启状态
- 关闭此开关 -> 整个卡片隐藏 + 重置触发状态
- 关闭后需重新连点 7 次才能让卡片重现

### 3. 启用调试 (总开关)
- 位置: 卡片内第一行
- 作用: 管理所有调试功能的主开关
- 逻辑: 总开关 ON + 副开关 ON -> 功能在软件中实际生效
- 总开关 OFF: 所有调试功能不生效

### 4. 新建/AI按钮模糊调试 (副开关 + 入口)
- 位置: 卡片内第二行
- 右侧: 副开关 + "进入"按钮
- 副开关: 控制该具体调试功能是否启用
- "进入"按钮: 点击进入三级调试页面
- 无底部详细说明文字

### 5. 三级调试页面
- 页面名称: "新建/AI按钮模糊调试"
- 返回按钮: 返回"更多设置"二级页面 (非设置主页)
- 参数控制: 弹性/模糊/饱和度/色散/通道分离/圆角/折射模式/浅色模式
- 参数自动保存到 localStorage

### 6. 实时预览窗口
- 背景: public/测试专用.png
- 大小: 长方形，约为卡片 1/3
- 按钮: 两个横向居中，X轴中间
- 按钮形状: 圆形 (LiquidGlass cornerRadius=999 + 内部 rounded-full)
- 按钮颜色: 无彩色，透明玻璃效果，图标 text-white/80
- 参考: https://liquid-glass.maxrovensky.com/

### 7. 首页按钮液态玻璃应用
- 位置永远不变 (fixed bottom-32 right-5)
- 启用调试后外观完全更新 (LiquidGlass 包裹)
- 使用 liquid-glass-react@1.1.1 默认参数

### 8. GlassSwitch prop 修复
- 所有 `<GlassSwitch>` 使用 `value={}` 而非 `checked={}`

## 开关层级

```
连点7次"更多设置"
  -> 高级调试卡片 (标题开关 ON)
       -> 标题开关 OFF -> 卡片隐藏，需重新触发
       -> 启用调试 [总开关]
            -> ON + 副开关 ON -> 应用到首页
            -> OFF -> 所有不生效
       -> 新建/AI按钮模糊调试 [副开关+进入]
            -> 副开关 ON -> 功能可被应用
            -> 进入 -> 调试页
```
